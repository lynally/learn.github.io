参考资料：http://www.jianshu.com/p/701b1095da11

hexo命令

######启动

  • hexo generate
  • hexo server
    ######hexo常用命令
  • hexo help #查看帮助
  • hexo init #初始化一个目录
  • hexo new “postName” #新建文章
  • hexo new page “pageName” #新建页面
  • hexo generate #生成网页，可以在 public 目录查看整个网站的文件
  • hexo server #本地预览，’Ctrl+C’关闭
  • hexo deploy #部署.deploy目录
  • hexo clean #清除缓存，
  • 强烈建议每次执行命令前先清理缓存，每次部署前先删除 .deploy 文件夹
  • hexo generate –deploy 以下两个都是生成+部署
  • hexo deploy –generate

######简写

  • hexo n == hexo new
  • hexo g == hexo generate
  • hexo s == hexo server
  • hexo d == hexo deploy
  • hexo g -d == hexo generate –deploy
  • hexo d -g == hexo deploy –generate
